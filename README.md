# NetoTest_bot — тренажёр английских слов (Telegram + PostgreSQL)

## Возможности
- Общий словарь (10+ слов) в PostgreSQL
- Тренировка: бот показывает слово на русском и 4 варианта на английском (кнопки)
- Правильный/неправильный ответ (при ошибке — попробовать снова)
- Добавление слова (персонально для пользователя)
- Удаление слова:
    - пользовательское слово удаляется из БД
    - общее слово скрывается только для данного пользователя

## БД
Скрипты:
- `sql/01_schema.sql` — создание таблиц
- `sql/02_seed.sql` — наполнение общими словами

Таблицы:
- `tg_user`
- `word`
- `user_hidden_word`

## Запуск
1) Создать БД `english_bot`
2) Выполнить `sql/01_schema.sql`, затем `sql/02_seed.sql`
3) Создать `.env`:
    - `TG_BOT_TOKEN=...`
    - `DATABASE_URL=...`
4) `pip install -r requirements.txt`
5) `python bot.py`

## Использование
- `/start` — начать тренировку
- `Дальше ⏭` — следующая карточка
- `Добавить слово ➕` — добавить персональное слово
- `Удалить слово 🔙` — удалить/скрыть последнее показанное слово
